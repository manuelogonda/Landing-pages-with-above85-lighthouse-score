# 🚀 QUICK REFERENCE CARD

## 5-Minute Setup

```bash
# 1. Get M-Pesa credentials (15 min)
https://developer.safaricom.co.ke/
→ Create app → Copy keys

# 2. Update config
Edit: backend/config.php
Lines: 11-14
→ Paste M-Pesa credentials

# 3. Create database
mysql -u root -p < backend/database.sql

# 4. Test without M-Pesa
Open: backend/demo.php
→ Create booking → Confirm payment

# 5. Deploy
Upload all files to web server
Set callback URL in Safaricom portal
Done! 🎉
```
